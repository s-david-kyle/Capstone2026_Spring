Popular graph database management system (GDBMS)
# Community Edition
Free to use, with following limitations
- Maximum database size of 10GB
- Number of concurrent connections is limited to 100
- Does not include support for graph analytics like PageRank and clustering
- Number of nodes limited to 100,000 with 1 million relationships
	- Third-party tools offer ways around this with graph partitioning, graph replication, and graph caching tools
	- Partitioning divides a graph into smaller graphs, which can be stored in separate Neo4j databases
- Supports only read-only transactions
- Graph algorithms are limited when compared to enterprise edition
