# Overview
Developed by Google, is a domain-specific LLM aimed at supporting clinical decision-making and medical information retrieval
- Built on Google’s Vertex AI platform, it employs advanced NLP techniques tailored for healthcare
- <mark>Limited to selected institutions and lacks an open-access web interface, while also raising concerns about handling sensitive data securely in online environments</mark>
