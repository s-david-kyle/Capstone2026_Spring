# Overview
Med-PaLM adapts the Flan-PaLM model to the medical domain using instruction prompt tun- ing instead of full-model fine-tuning. 
- <mark>Fine-tuning LLMs with hundreds of billions of parameters is computationally intensive</mark>
- This approach achieves state-of-the-art results in <mark>medical question answering, multiple-choice questions, and open-domain tasks, leveraging models with 8B, 62B, and 540B</mark> parameters from the PaLM and Flan-PaLM families
