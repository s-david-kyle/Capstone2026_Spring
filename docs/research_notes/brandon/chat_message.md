# Overview
The `ChatMessage` object in Streamlit is a simple class used to represent individual messages within a chat interface. It contains two attributes:

*   `role`:  This stores the role of the message sender (e.g., "user", "assistant").
*   `content`: This stores the actual text of the message.

Streamlit uses this object (or a similar structure) to display messages in a chat-like format, allowing you to represent both user input and bot responses within your application.  In the example above, it's a very basic demonstration, but it illustrates the fundamental structure.
