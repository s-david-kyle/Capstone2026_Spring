# Overview
A framework for learning **G**lobal-**L**ocal **R**epresentations for **I**mages using **A**ttention mechanism by contrasting image sub-regions and words in the paired report. Instead of relying on pretrained object detectors, we learn attention weights that emphasize significant image sub-regions for a particular word to create context-aware local image representations. Focuses on visualizing attention maps in radiographic images, aiding interpretability of model outputs.
